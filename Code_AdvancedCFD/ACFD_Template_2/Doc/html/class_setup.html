Detailed Description

Class for the simulation setup.

This class allocates and deletes all required pointers. Classes derived from Setup need to allocate objects of type Control, Grid, Time, Init, Boundary, Fluxes, Source, Update and Plot.

This class is not abstract but no object should be instantiated since there is only a default constructor that does nothing.
